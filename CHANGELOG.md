# Changelog

## v0.1.0

Initial public release of the `tangshan-yizhong-history` skill.

Included in this release:

- PDF text extraction based on `PyMuPDF`
- Local SQLite index with lightweight vector retrieval
- `query_index.py` for raw passage retrieval
- `ask.py` for retrieval-grounded direct answers
- `export_json.py` for structured JSON export
- `Makefile` shortcuts for install, query, ask, and JSON export
